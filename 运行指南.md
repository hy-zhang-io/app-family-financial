# 流体金融家庭财务APP - 运行指南

## 方式一: 使用 HBuilderX (推荐)

### 1. 下载并安装 HBuilderX
- 访问: https://www.dcloud.io/hbuilderx.html
- 下载"标准版"或"App开发版"
- 安装并启动 HBuilderX

### 2. 导入项目
1. 点击 HBuilderX 菜单: `文件` -> `导入` -> `从本地目录导入`
2. 选择项目目录: `g:\workspace\app\app-familyfinancial`
3. 点击"导入"

### 3. 运行项目

#### 运行到浏览器 (H5)
1. 在项目管理器中右键点击项目名称
2. 选择: `运行` -> `运行到浏览器` -> `Chrome`
3. 自动打开浏览器查看效果

#### 运行到小程序
1. 在项目管理器中右键点击项目名称
2. 选择: `运行` -> `运行到小程序模拟器` -> `微信开发者工具`
3. 需要先安装微信开发者工具

#### 运行到真机 (App)
1. 连接手机或启动模拟器
2. 在项目管理器中右键点击项目名称
3. 选择: `运行` -> `运行到手机或模拟器`

## 注意事项

### TabBar 图标
由于项目中配置了 TabBar 导航,需要在 `static/tabbar/` 目录下准备以下图标文件:

**必需的图标:**
- home.png / home-active.png
- transactions.png / transactions-active.png
- budget.png / budget-active.png
- assets.png / assets-active.png
- family.png / family-active.png

**临时解决方案:**
如果暂时没有图标,可以:
1. 修改 [pages.json](pages.json),删除 tabBar 配置
2. 或者使用纯色块临时替代

### 自定义组件路径
确保 [pages.json](pages.json) 中组件路径正确:
```json
"easycom": {
  "autoscan": true,
  "components": {
    "^fluid-(.*)": "@/components/fluid-$1.vue",
    "^ai-(.*)": "@/components/ai-$1.vue"
  }
}
```

## 常见问题

### Q: 提示找不到组件?
A: 确保 [pages.json](pages.json) 中的组件路径配置正确,或者使用 `easycom` 自动导入。

### Q: TabBar 不显示?
A: 检查 [static/tabbar/](static/tabbar/) 目录下是否有必需的图标文件。

### Q: Canvas 图表不显示?
A: 确保 Canvas ID 唯一,并在页面加载完成后调用绘制方法。
